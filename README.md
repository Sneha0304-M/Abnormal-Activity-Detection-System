# Abnormal-Activity-Detection-System
AI-powered abnormal activity detection system using Python, OpenCV, MediaPipe and Machine Learning.


## Overview
This project detects abnormal human activities using Computer Vision and Machine Learning techniques.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Machine Learning
- HTML
- CSS

## Features
- Real-time activity detection
- Human pose estimation
- Fall detection
- Abnormal behavior identification

## Applications
- Smart Surveillance
- Elderly Care Monitoring
- Public Safety Systems
